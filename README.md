# Maize Image Classification 🌽

A deep learning–based image classification project focused on detecting major foliar diseases in maize crops using computer vision and comparative disease modeling.

This project develops and evaluates separate deep learning models for three important maize diseases:

* Northern Corn Leaf Blight (NCLB)
* Southern Corn Leaf Blight (SCLB)
* Maize Downy Mildew (MDM)

The study includes:

* Image-based disease classification
* Disease-specific model development
* Training and validation of deep learning architectures
* Comparative disease prediction analysis
* Automated crop disease diagnosis

The project is designed to support early disease detection, reduce crop losses, and enable AI-driven precision agriculture applications for maize disease management.

Built using:

* Python
* TensorFlow
* Keras
* NumPy
* OpenCV
